# Review Action

1. Read `context/current-feature.md` to understand the target goals.
2. Review all code changes made in the workspace for this feature.
3. Check for:
   - ✅ Goals met
   - ❌ Goals missing or incomplete
   - ⚠️ Code quality issues or bugs
   - 🚫 Scope creep (code beyond goals)
4. Final verdict: State clearly whether it is "Ready to complete" or "Needs changes".
