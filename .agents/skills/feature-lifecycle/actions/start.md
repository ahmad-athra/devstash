# Start Action

1. Read `context/current-feature.md` - verify Goals are populated.
2. If empty, error out: "Run /feature load first"
3. Set Status to "In Progress" in the file.
4. Create and checkout the feature branch (derive name from H1 heading, e.g., `feature/add-navbar`).
5. List the goals back to the user, then implement them one by one.
